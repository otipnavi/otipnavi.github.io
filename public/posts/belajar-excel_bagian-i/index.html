      Belajar Excel, Bagian 1: Essentials

      Tentang

      Post ini merupakan catatan proses belajar Microsoft Excel Skills for Business saya lewat platform Coursera. Program ini dikelola oleh Macquaire University, Australia. Karena besarnya informasi yang saya pelajari, post akan dibagi menjadi empat post kecil sesuai pembagian course di Coursera: Essentials, Intermediate I, Intermediate II, dan Advanced.

      Latar Belakang

      Sebagai latar belakang, sudah setahun lebih saya tidak menggunakan Windows. Saya beralih ke ekosistem Linux, lebih spesifiknya EndeavourOS. Hal ini disebabkan karena:

      • Laptop saya tidak mendukung Windows 11,
      • Saya mempersiapkan penghentian bantuan untuk Windows 10 bulan Oktober 2025 ini,
      • Kampus menghentikan lisensi Microsoft Office di laptop saya beberapa minggu setelah saya lulus.

      Oleh karena itu, saya menggunakan program LibreOffice sebagai alternatif Microsoft Office. LibreOffice sudah cukup mumpuni untuk membuat dokumen-dokumen umum layaknya Microsoft Office. Hanya saja, ada beberapa fungsi yang tidak ada atau memiliki nama lain. Sebenarnya saya bisa-bisa saja menggunakan emulator Wine atau Office365 Online maupun Google Workspace, tapi saya yang lawas ini lebih suka menggunakan software yang jelas terinstal di perangkat saya.

      Microsoft Excel: Essentials

      Sudah cukup basa-basinya, saatnya beralih ke inti post ini, yaitu catatan hal-hal apa saja yang saya pelajari di Coursera. Bagian pertama, Essentials, mengajarkan saya hal-hal yang mendasar dan sebenarnya sudah pernah dipelajari di sekolah. Tapi tidak apa-apa, selain membantu me-refresh ingatan saya, pengajar juga memberikan banyak tips berguna.
